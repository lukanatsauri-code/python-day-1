# python-day-1

Day 1 of learning Python from absolute zero.

## Files

- `hello.py`
  - Version 1 prints "Hello, world"
  - Version 2 asks for your name and prints a greeting

- `numbers.py`
  - Asks for a number and tells you whether it is greater than 10

## Notes

This is part of Phase 1 (Self-Foundation): build technical literacy and public proof of work.
